# PRDs (Product Requirements Documents) - Summary

## Overview
This folder contains all Product Requirements Documents and feature-specific artifacts for the Advertising product line. Each sub-folder represents a distinct feature or feature group.

## Sub-folders

| Folder | Description | Key Focus |
|--------|-------------|-----------|
| [Feature Ads Reporting](./Feature%20Ads%20Reporting/) | Analytics and reporting for advertising performance | Dashboard templates, report types, profit optimization |
| [Feature APDC](./Feature%20APDC/) | Automated Performance-Driven Campaigns | Objectives, design challenges, RCA templates |
| [Feature Automated Bid Optimization](./Feature%20Automated%20Bid%20Optimization/) | Automated bid management and optimization | Bid synchronization, CVR estimation |
| [Feature Automated Campaign Generation](./Feature%20Automated%20Campaign%20Generation/) | Automatic campaign creation and structuring | Campaign structure, client implementations |
| [Feature Automated KWs Research](./Feature%20Automated%20KWs%20Research/) | Automated keyword research and classification | Prompts, R/S/C/N classification, UI mockups |
| [Feature Automated Pacing](./Feature%20Automated%20Pacing/) | Budget pacing and spend distribution | System design, modeling constraints |
| [Feature Automated Target ASINs Research](./Feature%20Automated%20Target%20ASINs%20Research/) | Competitor/related ASIN identification | Engineering feedback documentation |
| [Feature Campaign Config](./Feature%20Campaign%20Config/) | Campaign configuration interface | UI mockups for ASIN targeting |
| [Feature Miscellaneous](./Feature%20Miscellaneous/) | Miscellaneous feature documentation | Financial projection models |
| [Multi-Feature](./Multi-Feature/) | Cross-feature and platform-level PRDs | Ad SaaS Platform PRD (v1, v2) |

## Document Naming Conventions
- `Artifact_` - Supporting documentation and specifications
- `PRD_` - Product Requirements Document
- `Mockup_` / `Mockups_` - UI/UX design mockups
- `Supporting #_` - Supplementary materials for main documents
- `XXXAbortedXXX_` - Discontinued or superseded work
